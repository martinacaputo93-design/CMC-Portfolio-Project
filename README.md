## Progetti Inclusi

- **Accessibilità** — Studio delle linee guida WCAG e implementazione di design inclusivo
- **Discovery** — Ricerca utenti, personas, user journey map e analisi competitiva
- **Grafica** — Brand identity, loghi, palette colori e materiali di comunicazione
- **Copywriting** — Tone of voice, UX writing e strategia di contenuto

Tutti i progetti sono disponibili come PDF nella cartella `/assets` e in forma estesa su [Google Drive](https://drive.google.com/drive/folders/18Vj6PkvsY4ymgbH8TZR4krQnl2pCuIPq?usp=sharing).

## Funzionalità Tecniche

- Dark mode nativa con CSS custom properties
- Layout completamente responsive (mobile-first)
- Animazioni scroll-triggered con IntersectionObserver
- Menu hamburger accessibile da tastiera
- Form di contatto con validazione client-side
- Banner cookie GDPR con gestione localStorage
- Lazy loading immagini
- Effetto ripple sui pulsanti
- Navigazione attiva rilevata automaticamente
- Meta tag Open Graph per la condivisione social
